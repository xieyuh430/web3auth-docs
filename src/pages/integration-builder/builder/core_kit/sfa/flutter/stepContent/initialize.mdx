title: Create and initialize the Web3Auth SFA instance
---

Let's now create an instance of the `SingleFactAuthFlutter`.

We need the `network` to initialize SingleFactAuthFlutter class. `network` signifies the type of network(`mainnet`, `cyan`, `aqua`, or `testnet`) you
want to initialize web3auth with.

Read more about [SFA Flutter SDK](/sdk/core-kit/sfa-flutter).

:::
