To use Web3Auth, you'll need to create a project in the [Developer Dashboard](https://dashboard.web3auth.io) and get your client ID.

- Go to [Developer Dashboard](https://dashboard.web3auth.io/), create or select an Web3Auth project:

- Add `(bundleId)://auth` to **Whitelist URLs**.
